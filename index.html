Ollama setup help

Ollama must allow cross-origin requests from this page. Restart it with the OLLAMA_ORIGINS env var set, e.g.

OLLAMA_ORIGINS="*" ollama serve

On Windows, set OLLAMA_ORIGINS=* in System Environment Variables, then restart the Ollama service.

Ollama is optional: ranking works without it. Configure it only if you want LLM-generated rationales for each recommended reviewer.
